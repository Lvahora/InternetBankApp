# Ученая практика: Интернет-банк

## 1. Описание проекта
Данный проект представляет собой разработку WPF-приложения для управления банковскими счетами и осуществления финансовых операций в рамках системы интернет-банкинга.
Система предназначена для автоматизации процессов управления личными финансами клиентов банка: просмотр счетов, выполнение переводов, оплата услуг, управление банковскими картами и кредитными продуктами, а также формирование отчётности по операциям.
Тип работы: Выпускная квалификационная работа (дипломный п

## 2. Цель и задачи проекта
Цель: разработка безопасного и функционального WPF-приложения для управления банковскими счетами и переводами.
Задачи:
Анализ предметной области банковских операций и требований к безопасности
Проектирование реляционной базы данных (7 сущностей)
Реализация бизнес-логики: переводы, проверка баланса, расчёт комиссии, формирование выписок
Разработка интуитивного WPF-интерфейса с поддержкой MVVM-паттерна
Написание модульных тестов для критически важных операций
Обеспечение целостности данных и транзакционной безопасности
Область применения: розничный банковский сектор, системы дистанционного банковского обслуживания (ДБО).

## 3. Функциональность
Банковский счёт
• Транзакция
• Кредитный калькулятор
• Перевод средств
• Проверка баланса
• Расчёт комиссии
• Формирование выписки
• Оплата услуг
• Расчёт кредита

## 4. Архитектура проекта
Тип архитектуры: Клиент-сервер с локальной БД
**Технологии:**
• Язык: C# (.NET 6/7/8)
• Фреймворк: WPF (Windows Presentation Foundation)
• ORM: Entity Framework Core
• База данных: SQL Server Express / PostgreSQL
• Тестирование: xUnit / NUnit + Moq
• Валидация: FluentValidation
• Логирование: Serilog / NLog
• Сборка: MSBuild, NuGet

## 5. Установка и запуск
• ОС: Windows 10/11 (x64)
• .NET Desktop Runtime 7.0 или выше
• SQL Server Express LocalDB / PostgreSQL 14+
• Visual Studio 2022 (рекомендуется) или Rider

## 6. Использование
Главное окно включает:
• Боковое меню навигации (Счета | Переводы | Платежи | Карты | Кредиты | Выписки)
• Область отображения контента (динамическая загрузка View)
• Статус-бар (текущий пользователь, баланс, уведомления)

## 7. Результаты и выводы
В ходе выполнения дипломного проекта была разработана полнофункциональная система интернет-банка, которая:
• Обеспечивает безопасное управление финансовыми операциями клиента
• Автоматизирует рутинные расчёты (комиссии, кредитные платежи)
• Предоставляет удобный интерфейс для повседневного использования
• Поддерживает расширение функционала за счёт модульной архитектуры
